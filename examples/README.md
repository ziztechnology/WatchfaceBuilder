# Examples

This directory contains example watchfaces demonstrating different features and templates.

## Structure

Each example contains:
- `build.sh` - Build script
- `README.md` - Description of the example
- Source files (if applicable)

## Running Examples

### Simple Digital Clock

```bash
cd examples/simple
./build.sh
```

### Analog Clock

```bash
cd examples/analog
./build.sh
```

### Digital Clock with Effects

```bash
cd examples/digital
./build.sh
```

### Custom Template

```bash
cd examples/custom
./build.sh
```

## Adding Your Own Example

1. Create a new directory in `examples/`
2. Add your custom HTML/CSS/JS files
3. Create a `build.sh` script
4. Add a `README.md` describing your example
5. Submit a Pull Request
